## 💻 주요 명령어

### 📦 의존성 설치
```bash
sudo apt update
sudo apt install cmake libsfml-dev
```

### ⚙️ 빌드 및 실행
```bash
./build.sh          # CMake로 빌드
./build/main        # 실행
```

### 🧹 빌드 폴더 삭제
```bash
rm -rf build
```
